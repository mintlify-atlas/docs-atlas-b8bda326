## What is omegaUp?

omegaUp is a **free educational platform** that helps improve programming skills through competitive programming challenges. Used by tens of thousands of students and teachers throughout Latin America, omegaUp provides a comprehensive environment for learning, practicing, and competing in algorithmic problem-solving.

Whether you're a student looking to sharpen your coding skills, a teacher organizing contests for your classroom, or a competitive programmer preparing for olympiads, omegaUp offers the tools and community you need to succeed.

## Architecture overview

omegaUp follows the **Model-View-Controller (MVC)** design pattern:

- **Model**: Data Access Objects (DAOs) and Value Objects (VOs) manage database schemas and business logic
- **View**: Vue.js components and templates render the user interface
- **Controller**: PHP controllers implement the REST API and handle business logic

The platform includes several key components:

- **Frontend**: Web application serving users, teachers, and administrators
- **GitServer**: Version control for problem statements and test cases
- **Grader**: Distributed system for evaluating code submissions
- **Runners**: Isolated execution environments using Linux containers and seccomp-bpf
